# 幸存偏误
> 为什么你该去逛逛墓地

系统性高估了成功概率

```
多分析失败的案例，总结失败原因
```

# 自我幻觉
> 哈佛是好大学还是烂大学？我们不清楚

混淆了选择标准与结果

```
需要明确什么是因，什么是果，影响结果的根本原因是什么
```

# 过度自信效应
> 你为什么会系统性地高估自己的学识和能力

系统性高估自己的学识和能力

```
别相信“好像”、“还可以”等猜测及无证据依托的预测，在筹划任何事情时，都考虑异常、悲观场景，做好最坏的打算

```

# 从众心理
> 就算有数百万人声称某件蠢事是对的，这件蠢事也不会因此成为聪明之举

只要别人做什么，我也跟着做什么，我的行为就是正确的。越多的人认为一个想法是正确的，这个想法就是更加正确

```

认可一件事情，一定是因为自己经过思考之后去认可的，而不是因为别人认可了

```

# 纠缠与沉没成本

> 你为什么应该忽视过去

放不下“已经付出了这么多”，放弃就等于投降

```
立于当前的形势及未来的评估来思考下一步

```

# 互惠偏误
> 你为什么不该让别人请你喝饮料

大家都觉得应该要遵守礼尚往来

```
无功不受禄，得审视眼前的每一个小便宜
```

# 确认偏误
> 遇到 “特殊情况” 这个词，你要格外小心

逃避“特殊情况”，主观过滤与自己观点相矛盾的信息，趋向让自己认可的解释一直成立

```
发现与自己理论想矛盾的信息需要刻意记录下来，要思考为什么或出现“特殊情况”
换位思考，客观审视自己的理论

```

# 权威偏误
> 你为什么该藐视权威

盲从专家理论

```
除非找不到证据说明它是错的，否则就不能相信
```

# 对比效应
> 你为什么最好别找模特儿等级的朋友一起出门

好与坏，建立在对比的基础上，同一样东西，它的参照物不同，结论也就不一样

```
在衡量价值前，必须前给出基准线，确认参照物的价值

```

# 现成偏误
> 你为什么宁可用一张错误的地图，也不愿意没有地图

依据现成的例子来建立或佐证自己的观点理论

```
接受不同的想法，加以客观思考
```


# 否极泰来论（确认偏误的变体）
> 如果有人建议你选择一条 “先经历痛苦的道路”， 你应该敲响警钟

在好转之前，会先恶化

```
制定的方案及计划，需要明确的可以检测的里程碑，阶段性验证进度是否正常，是否健康地完成每一个目标，确认最终目标是没问题的

```


# 故事偏误

用故事扭曲和简化现实


```
以无关联的眼光来看每个故事，并且挖掘每个故事背后本身的缘由

```

# 事后诸葛偏误|hindsight bias

事后回归时一切都显得是可以理解的、不可避免的

```
没有预言家，世界的未来是不可预测的
```

# 控制错觉

你以为大部分的事情都在自己的控制之内

```
你实际控制的很可能比你以为的要少
只关注最重要的事情

```

# 激励过敏倾向

人会倾向做对自己有益的事情
一旦有激励加入游戏或改变了激励，人们就会迅速而剧烈地改变自己的行为
人们是对激励做出反应，而不是对激励背后的目的做出反应

```
如果某个人或组织的行为发生了变化，而该变化不符合寻常，背后可能隐藏着激励机制
在执行激励机制前，需要明确要解决的核心问题是什么，激励应与目的联系起来，单纯的激励会导致问题恶化

```


# 回归均值

长远来看，同一件事情的结果，会围绕着均值摇摆

```
在分析结果时，不可忽视回归均值的可能性，多加验证，避免对核心价值的误判

```


# 公地悲剧

在放任自由的情况下，公共资源最终将被剥削殆尽

```

将公共资源私有化，如果无法私有化，必须加强管理

```


# 结果偏误|史学家错误
> 切勿以结果判断决定

倾向于以结果判断决定，而不是当时做决定的过程

```
避免以结果来判断决定，更换决策，必须要有依据，在毫无依据可支撑的情况下，宁愿不要做任何变化

```

# 选择的悖论
> 为什么更多反而是更少

过多的选择会降低生活质量

* 选择范围大会导致无所适从。选择越多，越难做决定，最终什么都不做。
* 选择范围大会导致作出更差的决定。选择越多，面临的决策因素就越复杂，人们总是逃避复杂，容易简单的将偏好最大化，来作为核心选择条件。
* 选择范围大会导致不满。选择越多，在选择后就越没把握，因而也就不满

```

在选择前，先要仔细考虑你想要什么，将每一项标准列出来，并务必遵守它们
一旦选择，就满足并往前走，不要回头看

```

# 讨喜偏误

> 你行为不理性，是因为你想讨人喜欢

某人越讨喜，我们就越倾向于帮助他
* 外表有吸引力
* 在出身、个性和兴趣上与我们相似
* 他觉得我们讨人喜欢

```
在做决策时，需要将影响你决策的角色置于陌生位置，尽可能处于客观或更严厉的角度来看待问题
必须像机器般地理性审视问题
拆解核心逻辑，明确每个核心逻辑的决策条件

```

# 禀赋效应

> 请不要死抱着某种东西不放

本性是倾向利好自己的，当我们出售某物时，我们要求的钱多于我们自己愿意为它支付的钱

```
不要盲目比较得失，并不是所有的事情都是必然的，它会随着自己不同的选择，而不一样，切忌结果论
要明确核心诉求，不能止步于当前得失

```

# 奇迹
> 不可能事件的必然性

意外是少概率事件，但不是绝对不发生

```
墨菲定律，做好准备
```

# 团体迷思
> 共识为什么有可能是危险的

从众心理，会导致盲从

```
独立思考，勇于表达自己的思想
珍惜唱反调的意见，可能真理被掌握在少数人手上

```

# 忽视概率偏误

> 累计奖金为什么会越来越多

缺少对概率的直觉理解，会导致作出错误的决定

```
除非风险为0，否则必须计算风险

```

# 零风险偏误
> 你为什么会为零风险支付过多

愿意付出更大的代价，追求绝对的安全

```
不存在完美无缺的、绝对安全的结果，要做好坏的打算并且做好准备
坦然接受一切

```

# 稀少性谬误
> 为什么饼干越少越好吃

稀少度本身成为了一种价值

```
需要仅按实际作用来判断一样东西的价值

```

# 忽视基本概率

> 当你在怀俄明州听到马蹄声、见到黑白条纹时......

越精确的描述，约能引导人们偏离统计真相

```
做决策或定目标时，不要忽视了基本概率，要基于历史数据的统计而得出评估

```


# 赌徒谬误

> 为什么没有一种平衡命运的力量

以为命运有一种平衡力量，即使可能性很小的事件，终将都会发生的，并且就在下一次

```

现实中的事件大部分都是相互联系的，如果你面对的事情是独立事件，那它不适用于回归均值，得依靠统计基础概率的思维来思考

```


# 锚定效应

> 数字轮盘如何搞得我们晕头转向

我们会以熟悉的东西，去度量或评估陌生的东西

```

一个对象的价值越无法确定，就越容易受锚定影响
谁来主动设定锚，事情发展的主导权大多数就会在谁的手上

```


# 归纳法

> 如何把别人的钱弄进自己的口袋

由观察个体得出普遍适用的结论的倾向

```
推理需要合理的依据，该依据可以通过反面例子来佐证，需要确认的是，这个反面例子是普遍存在，还是罕见的，以用来验证结论是否普遍适用
表象往往对我们起到的是引导作用，而到决策环节，依靠的是必然是背后的逻辑


```

# 规避损失

> 为什么凶恶的面孔比友善的面孔更容易引起我们注意

在我们的心里，损失要比同样多的收益引起的影响更大，人们害怕失去某种东西的想法要比获得某种同等价值的东西的想法强烈

```

是引发”纠缠与沉没成本“偏误的底层逻辑，对于得失的衡量要基于理性（数据统计及逻辑）的分析

```

# 社会性懈怠

> 团队为什么会使人慵懒

在团队中个人的效率无法直接看到，而是与团队融合在一起的，出于理性的思考：假如使出一半力就行，又不会引起注意，那为什么要使出全力呢？

```
团队的效率在自然放纵的情况下，是存在效率天花板（最大的慵懒度），达到上限后，继续加人，带来的是成本的增加，因为团体效率不增，人均效率会越来越低，维护团队规模的成本就只会增加，而得不到同等的回报。
要提升团队效率，需要激励机制，来彰显个人效率，营造竞争氛围。

```

# 指数增长

> 一张对对折的纸为什么会超过我们的想象

指数增长意味着翻倍、递增，随着时间，结果会超于惯性地被放大，而非线性增长

```
涉及到增长率时，不要相信直觉，要拿计算器出来算一下

```


# 赢家的诅咒

> 你愿意为100欧元支付多少钱

拍卖的赢家大多是事实上的输家，因为他是支付得最多的，也就是这笔支付是最有可能超越真实价值的

```

按自己的逻辑对目标定价，可以定一个最高价格（溢价20%），并且遵守它，不要受环境影响去抬升它

```


# 基本特征谬误

> 千万别问一位作家他的小说是不是自传

系统性地高估人的影响，在解释某些东西时低估情境因素

```

在分析问题时，需要提醒自己更加关注客观因素或者问题本身，而不要只关注直接传递信息给自己的人或事情

```

# 错误的因果关系

> 你为什么不该相信仙鹤送子

往往来源于传说的、未经过实践证实的因果关系，是错误的

```
两者之间有相互关系，但是不等于有因果关系
未经过实践证实的结论，不要轻易相信

```

# 光环效应
> 长相好的人为什么容易事业有成

我们让某一个方面照花了眼睛，并由此推导（猜测）全貌，而不去进行更仔细的研究分析

```
避免被光鲜的表象蒙骗，要主动深入挖掘出有异常、有缺陷的东西，因为一件事情它是不存在完美的，所以它唯有具备好与不好的两面，才是相对完整的全貌

```


# 替代途径

> 恭喜你赢了俄罗斯轮盘赌

所有导致同样结果的可能发生但没有发生的事情

```

做选择时，请时刻考虑有什么样的替代途径，选择风险最小的途径来达成目标

```

# 预测的错觉

> 水晶球如何歪曲了你的目光
>> 预言未来的人有两种：一种是一无所知的人，另一种是不知道他们自己一无所知的人

预言越多，纯属巧合地猜中的概率就会上升

```

对预测持批判态度

```


# 关联谬误

> 有说服力的故事为什么会误导人

人们更愿意相信与故事背景有关联的结论

```

人存在两种思维，1）直觉、机械、直接的“直觉思维”；2）有意识、理性、缓慢、有逻辑性的“意识思维”。当面对问题时，直觉思维会先作出反应，请稍微等等，看看意识思维给出的答案是什么，再作决策

```


# 框架效应

> 言为心声

视表达方式的不同，我们会对同样的事情做出不同的反应

```
要意识到，有框架效应的存在，它会左右我们对信息的认知，如果涉及到决策，请遵循自己的逻辑分析与实践经验

```

# 行动偏误

> 为什么不行动光等待是种痛苦
>> 人类的全部不幸就是他们不能安静地待在他们的房间里

即使毫无用处，也要采取行动

```
遇到事情，在自己分析出形势之前，即自己看来目前的形势不明、矛盾、看不透时，请不要采取任何行动

```

# 不作为偏误

> 为什么你不是答案就是问题
>> 如果你不解决问题，那你等于就是在延续问题、加重问题，当然你就成为了“问题的一部分”

在我们判断无论放弃还是行动都会带来损害的情况下，倾向于选择放弃

```
遇到问题，在你明确了目前形势之后，请理性衡量成本（风险）及收益，必须要给出符合自己逻辑的解决方案，而不是无脑的不作为、不解决

```


# 自利偏误

> 你为什么从不自责

成功归于自己，失败归于外因

```
珍惜批评自己的朋友，要重视逆耳的意见
要先从自己身上找问题，或者不妨认为自己一定是有问题
```

# 享乐适应症

> 你为什么应该缩短上班路程

大部分的幸福，我们都是比较快适应的，适应后，就没有幸福感了，相对的会系统地高估负面情绪的持续时间和强度

```
避免长时间也不会习惯的负面效应（视个人的价值观而定）

不要对物质期待太高，且只有短期效果

做自己最爱做的事情


```

# 自我选择偏误

> 请不要惊讶有你存在

以为自己就是特别的

```

请结合“基本概率”来审视自己的处境

```

# 联想偏误
> 为什么经验有时让人变蠢
>> 我们应该注意，一个经历里隐藏着多少智慧，我们就只吸取多少，不要多；好让我们不像做过热灶台的猫一样。被烫过的猫永远不会再坐到热灶台上去，这是对的；但它也永远不会再坐到冷灶台上去了

出于巧合而积累的经验，会让人们倾向于将彼此毫无关系的东西联结在了一起

```

要理性分析经验，不全都是合理的，只吸取有智慧的经验，只有符合逻辑、符合事实的知识，才能称得上智慧
做决策，不能仅依靠经验，还得参考我们最不愿意接触的负面情报
提醒自己要小心 “斩来使综合征”（信使与消息内容被联系在了一起，由于不喜欢坏消息，所以也不喜欢带来坏消息的人）


```

# 新手的运气

> 假如开始时一切顺利，请务必多加小心

延续联想偏误，错误地与从前的成功建立联系，那成功只是运气

```
成功的时候，不要着急作出判断。有以下建议：
1）如果你长期比其他人成功，那可能是能力的作用，但绝不能过于自信、自满
2）参与的人越多，自己由于纯粹的运气长期成功的概率就越大
总的来说，胜不骄

```

# 认知失调

> 你如何撒点小谎，让自己感觉好一些
>> 吃不到葡萄说葡萄酸

自己当初的打算与结果不符，事后作出别的解释来自欺欺人

```

对自己诚实

```

# 双曲贴现
> 及时行乐，但请只限于星期天

一个决定离现在越近，我们的“情感利息”就越多，因此随着时间长度的不同，对于同一件事作出的决定是不一致的

```
控制及时行乐的冲动

```
